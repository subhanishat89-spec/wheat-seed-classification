# Wheat Seed Variety Classification Using Deep Learning

This project presents a comparative study of multiple CNN architectures for classifying wheat seed varieties using a small RGB image dataset.

## Models Used
- EfficientNetB0
- ResNet50
- VGG16
- Custom CNN
- Attention CNN (SE Block)

## Best Model
Attention CNN achieved the best performance:
- Accuracy: 78.12%
- AUC: 0.9197

## Project Structure
- code/ → all implementation files (Tasks 1–4)
- reports/ → individual and final reports
- presentation/ → final presentation slides

## Dataset
- 375 RGB images
- 3 classes: Akbar-19, Dilkash-20, Urooj-22

## Tasks
- Task 1: Data analysis
- Task 2: Pre-trained models
- Task 3: Custom CNN
- Task 4: Attention CNN

## Evaluation Metrics
Accuracy, Precision, Recall, F1-score, AUC, Confusion Matrix, Paired t-test
